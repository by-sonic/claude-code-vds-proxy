# Changelog

All notable changes to this project are documented here.

## 0.1.0 - 2026-07-05

- Initial public release.
- Private Squid CONNECT proxy bound to VDS loopback.
- Persistent macOS SSH tunnel and launchd proxy environment.
- Host-based fail-closed controls for core Claude destinations.
- Exit-IP, TLS, direct-route and VDS CONNECT-log verification.
- Claude Code binary endpoint audit and update lifecycle manager.
- English and Russian documentation.
