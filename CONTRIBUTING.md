# Contributing

Thanks for helping improve Claude Code VDS Proxy.

1. Fork the repository and create a focused branch.
2. Keep changes compatible with the macOS system Bash where practical.
3. Never commit real IP addresses, usernames, SSH key paths or credentials.
4. Run `./tests/static.sh` before opening a pull request.
5. Explain network and security behavior changes in the pull request.

Bug reports should include the macOS version, VDS distribution, Claude Code
version, installation method and redacted verification output.
